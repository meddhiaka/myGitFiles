# arrow function simple transformation



### * the simple function

```javascript
function fn(x){
    // c du blabla
}
```

## * the arrow function after conversion 

```javascript
(x) => {
    // c du blabla
};
```





<hr>

### * to reuse an arrow function just :

```javascript
const newFn = (x) => {
    // c du blabla
}
```





<hr>

### a real example

```javascript
function square(a){
    return a*a;
}
```

#### to

```javascript
const square = (a) => {
    a*a
}
```
